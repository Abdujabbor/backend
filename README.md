# Imlo Backend

Имло лойиҳасининг бэкенд қисми

## Керакли муҳит

  * Python==3.7
  * PostgreSQL==12.0


## Керакли кутубхоналар

  * FastAPI==0.33.0


## Лойиҳани ўрнатиш

```bash
$ make install
$ make download
```


## APIдан фойдаланиш тартиби

```bash
$ make run
```


## Тест қилиш тартиби

```bash

```
